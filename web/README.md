# Horizen Gym — Back Office

Admin web app for managing members, subscriptions, plans, and features.

## Setup

```bash
npm install
cp .env.example .env  # optional; defaults to http://localhost:3001
```

## Run

```bash
npm run dev
```

Open http://localhost:5173. Log in with an admin account (e.g. admin@gym.com). The API must be running on port 3001.

## Build

```bash
npm run build
```

Output in `dist/`.
